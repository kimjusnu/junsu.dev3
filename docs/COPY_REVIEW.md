# 카피/문구 점검 요약 (2026-02-19)

아래는 전체 화면 문구를 훑어보며 **수정 또는 개선하면 좋아 보이는 포인트**를 정리한 것입니다. 실제 수정은 아직 적용하지 않았고, 제안 목록만 작성했습니다.

## 1) 최우선 이슈: 문자 깨짐(모지바케) 다수
여러 컴포넌트에서 한글이 깨져 보입니다. 예) `?쒕퉬???`, `源以??`, `??/span>` 등

- 원인 추정: 파일이 UTF-8이 아닌 인코딩으로 저장되었거나, 인코딩 변환 과정에서 깨짐
- 해결 권장:
  - 에디터에서 해당 파일들을 **UTF-8로 재저장**
  - 정상 한글로 복원 후 문구 개선 진행

깨짐이 복구되지 않으면, 아래 제안 문구를 그대로 넣어도 다시 깨질 수 있습니다.
